Данное приложение позволяет наложить картинку BMP на каждый кадр видеоряда YUV420.
Данное приложение можно использовать в Windows и Linux.

# Использование

Запустите приложение со следующими параметрами:  
*filename1*: имя BMP файла  
*filename2*: имя YUV файла  
    *width*: ширина в пикселях YUV файла  
   *height*: высота в пикселях YUV файла  
   *frames*: количество кадров в YUV файле  

Поскольку изображения формата YUV не содержат информацию о файле (такую, как в BMP), то параметры width, height и frames являются обязательными.
Пример команды: C:\bmp2yuvCover.exe example.bmp example.yuv 352 288 300
Можно прописать полные пути до файлов, а можно прописать имена файлов, лежащих в той же директории, что и исполняемый файл. Примеры файлов с изображениями лежат в репозитории по пути ./examples.
